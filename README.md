# **Data Analytics Virtual Experience Program by KPMG**

## Project Description
The main goal of this project is to work on real data based on a real life business problem. 

This programme covers all of the key skills of a data analytics consultant:

-Data Quality Assessment
-Data Insights
-Data Insights and Presentation

### Methods used

* Data Cleaning
* Machine Learning
* Data Visualisation

### Technologies used

* Python
* Excel
* Pandas, Numpy, Jupyter Notebook
